waxe
====

Haxe + wxWidgets

A fork from [nmehost/waxe](https://github.com/nmehost/waxe).

###Building 

```
git clone git@github.com:nmehost/waxe-works.git
cd waxe-works
haxelib dev waxe-works .
lime rebuild waxe windows
```
